RepoGrabberGrapher

This tool is designed to help dynamically visualize statistics from a provided dataset generated by the tool 'RepoGrabber.'

It takes input of a RGDS dataset from RepoGrabber run and automatically imports the dataset into a local SQLite DB, then runs queries (that are cached once ran once.)
